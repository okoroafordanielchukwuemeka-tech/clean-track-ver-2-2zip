# CLEAN TRACK — SMART ORDER CREATION + FULL SETTINGS & AUTOMATION HUB

IMPORTANT:

Clean Track is a multi-tenant B2B Laundry SaaS platform.

The goal of this phase is to:

* make order creation extremely fast
* reduce worker mistakes
* centralize business configuration
* create a true operational control center for each laundry
* prepare for future WhatsApp automation
* prepare for future push notifications

IMPORTANT:

The system should remain:

* simple
* fast
* mobile friendly
* operationally efficient
* easy for non-technical laundry owners and workers

A worker should be able to create an order in under 60 seconds.

DO NOT over-engineer the interface.

---

# PART 1 — FULL SETTINGS & AUTOMATION HUB

Create a dedicated Settings page.

Only Owners/Admins can access it.

Settings should be grouped into clean sections.

# PART 4 — SMART ORDER CREATION SYSTEM

Redesign order creation completely.

Goal:

Fast.
Simple.
Error-proof.

---

## CUSTOMER STEP

Allow:

Select Existing Customer

OR

Create Customer Instantly

without leaving the order screen.

If customer exists:

* Name
* Phone
* Address

## ITEM SELECTION STEP

Workers should select services directly.

3 Trousers

1 Duvet

1 Rug
