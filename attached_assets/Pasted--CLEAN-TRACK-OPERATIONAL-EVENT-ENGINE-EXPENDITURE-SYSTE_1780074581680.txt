* operational automation

# PART 1 — OPERATIONAL EVENT ENGINE

## 1. Create Central Event Architecture

Design a scalable event system that reacts automatically to:

* new orders
* worker assignment
* overdue orders
* due-soon orders
* order ready
* payments received
* unpaid balances
* partial pickups
* completed pickups

IMPORTANT:
This event architecture should later support:

* browser push notifications
* WhatsApp notifications
* SMS
* email

without major rewrites.
