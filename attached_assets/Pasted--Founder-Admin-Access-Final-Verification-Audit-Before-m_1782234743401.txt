# Founder Admin Access Final Verification Audit

Before moving to Paystack integration, perform a complete founder admin access verification.

The goal:

Ensure the founder has a clean, secure, and usable admin workflow separate from laundry owners and workers.

DO NOT build new features yet.

First audit the current implementation.

---

## PHASE 1 — Admin Account Creation

Verify:

1. How are founder admin accounts created?

Check:

* Is there a signup page?
* Is it database seeded?
* Is it created manually?
* Is there a CLI/script?

Document the exact current process.

If no safe creation method exists:

Create a secure founder admin seed script.

Requirements:

Create:

Founder Admin

Fields:

* name
* email
* password hash
* role = super_admin

DO NOT expose admin creation publicly.

---

Can the founder:

Open the application

↓

Access admin dashboard
